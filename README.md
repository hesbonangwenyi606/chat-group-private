## React + TypeScript + Vite
This is a minimal starter template using React, TypeScript, and Vite. It includes fast development with HMR, basic ESLint rules, and support for Tailwind CSS if added.

## Features
**Blazing-fast Vite dev server**
**TypeScript support**
**Hot Module Replacement (HMR)**
**Tailwind CSS-ready**
**ESLint for clean code**
**Simple and clean folder structure**

### Getting Started
Clone the repository
git clone https://github.com/hesbonangwenyi606/<your-repo>.git
cd <your-repo>

### Install dependencies
npm install

### Run the development server
npm run dev

Open in browser: http://localhost:5173

## Project Structure
/
├── public/              # Static assets
├── src/                 
│   ├── components/      # UI components
│   ├── App.tsx          # Root component
│   └── main.tsx         # Entry file
├── index.html           # Main HTML file
├── vite.config.ts       # Vite config
├── tsconfig.json        # TypeScript config
└── tailwind.config.ts   # Tailwind config (if enabled)

## NPM Scripts
Command	Description
npm run dev	Start dev server
npm run build	Build production files
npm run preview	Preview production build
npm run lint	Run ESLint

## Production Build
npm run build
Build output will be in the dist/ folder.
